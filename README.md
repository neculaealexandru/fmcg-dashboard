markdown# 🛒 Romania FMCG Market Analytics Dashboard

Interactive dashboard analyzing Romania's fast-moving consumer goods market from 2010 to 2024.

## 🔗 Live Demo
[fmcg-dashboard.streamlit.app](https://fmcg-dashboard-bni8qhj35aypzabrqxmpty.streamlit.app/)

## 📊 Features
- Market size & category breakdown (Food, Beverages, Personal Care, Confectionery)
- Consumer behaviour trends — monthly spend & online channel growth
- Private label penetration vs consumer confidence index
- Regional market intelligence across all 8 Romanian development regions

## 🛠️ Built With
- Python · Streamlit · Plotly · Pandas

## 📁 Data Sources
- Eurostat — European consumer market statistics
- INS (Institutul Național de Statistică) — Romanian national data
